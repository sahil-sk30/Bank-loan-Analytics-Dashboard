# Bank Loan Analytics Dashboard

## Project Overview
This project analyzes 38K+ loan applications to identify approval trends, customer segments, and lending performance.

## Tools Used
- Excel
- SQL
- Power BI
- Tableau

## Key Insights
- Loan approval trend analysis
- Customer segmentation
- KPI dashboard creation
- Lending performance tracking
